IO流（二）
java IO流

字节流复制文件

1.每次复制1字节

InputStream fis = new FileInputStream("要复制的文件的路径");
OutputStream fos = new FileOutputStream("复制到的路径");
int b = -1;
//循环读写
while((b = fis.read())!=-1){
    fos.write(b);
}
//关闭流，先用的后关
fos.close();
fis.close();

2.使用字节数组

InputStream fis = new FileInputStream("要复制的文件的路径");
OutputStream fos = new FileOutputStream("复制到的路径");
byte[] bs = new byte[1024];
int len = -1;
while((len = fis.read(bs)!=-1){
    fos.write(bs,0,len);//注意，如果这里不指定len，会出现乱码问题
}
fos.close();
fis.close();

为什么不指定打印的len会出现乱码问题？

答：假如文档中的是中文的话，复制过来就会出现乱码问题，GBK中一个中文对应两个字符，使用字节流会把中文拆分成对应的字节后，再传输。但是因为每次都只传输1024个字符（期间的数字，空格等算一个字符），汉字从文件中拆分成字符，然后再写入，将字符组合成中文的时候，可能会出现字符拼接位置不对，造成乱码。

字节缓冲流

字节缓冲流操作文件，缓冲流不能直接访问文件，必须要借助字节流访问！

InputStream fis = new FileInputStream("操作的文件");
BufferedInputStream bis = new BufferedInputStream(fis);
//创建字节数组
byte[] bs = new byte[fis.available()];
//将文件读到数组中
bis.read(bs);
System.out.println(new String(bs));
//关流
bis.close();
fis.close();

字节缓冲流复制文件

1.使用带缓冲区的字节流，每次1字节复制文件

BufferedInputStream bis = 
    new BufferedInputStream(new FileInputStream("要复制的文件路径"));
BufferedOutputStream bos = 
    new BufferedOutputStream(new FileOutputStream("复制过去的路径"));
//循环读写
int b=-1;
while((b = bis.read())!=-1){
    bos.write(b);
}
//关流
bos.close();
bis.close();

2.使用带缓冲取得字节流，使用字节数组复制文件

BufferedInputStream bis = 
    new BufferedInputStream(new FileInputStream("要复制的文件路径"));
BufferedOutputStream bos = 
    new BufferedOutputStream(new FileOutputStream("复制过去的路径"));
//循环读写
//使用字节数组
byte[] bs = new byte[1024];
int len;
while((len = bis.read(bs))!=-1){
    bos.write(bs,0,len);
}
//关流
bos.close();
bis.close();

总结

带缓冲区的效率高区不带缓冲区的，字节数组的效率高于单字节复制的效率

字符流

  • 字符输入流：InputStreamReader（外界—>程序）
  • 字符输出流：OutputStreamWriter（程序—>外界）

在使用字符流时，也需要经过FileInputStream来对文件进行操作

字符输入流InputStreamReader

InputStreamReader isr = 
    new InputStreamReader(FileInputStream("操作的文件路径")); 
//字符流中的read与字节流一样，也有三种参数形式
//1.一次读一个字符
//2.一次读一个字符数组
//3.一次读一个字符数组，指定开始位置与长度
//=====每次读取一个字符=============
int b = -1;
while((b = isr.read())!=-1){
    System.out.println((char)b);//int型强转成char
}
isr.close();
//=====每次读取一个字符数组==========
int len = -1;
char[] c = new char[3];
while((len = isr.read(c))!=-1){
    System.out.println(new String(c));
}
isr.close();
//=====每次读取一个字符数组，指定数组的位置与长度=======
int len = -1;
char[] c = new char[3];
while((len = isr.read(c))!=-1){
    System.out.println(new String(c,0,len));
}
isr.close();

字符输出流OutputStreamWriter

OutputStreamWriter osw = 
    new OutputStreamWriter(new FileOutputStream("操作的文件路径"));
//向文件中写入字符
osw.write('哼');
//清空缓冲区
osw.flush();
//直接写入字符串
String str = "我是字符串";
osw.write(str);
osw.write(str,0,2);//也可以指定位置与长度
osw.write(65);//ASCII码
//关闭流
osw.close();

字符流设置编码格式

常见的编码格式

  • Unicode：支持世界上所有语言的编码，UTF-8，UTF-16
  • GBK：简体中文
  • BIG5：繁体中文
  • ASCII：美国国家信息标准码
  • ISO-8859-1：拉丁码

在文件流中，使用相同的码表进行编码和解码才不会出现乱码

//设置写入的编码
OutputStreamWriter osw = 
    new OutputStreamWriter(new FileOutputStream("路径","编码格式"));
osw.write("哈哈哈哈");
osw.close;
//设置读取得编码格式
InputStreamReader isr = 
    new InputStreamReader(new FileInputStream("路径"),"编码格式");
int b = -1;
while((b = isr.read())!= -1){
    System.out.println((char)b);
}
isr.close();
//此时，用这种方法读取，都会输出正常文字，只要保证输入输出的编码格式保持一致

字符输入输出流（FileReader，FileWriter）

  • FileReader：inputStreamReader的子类
  • FileWriter：OutputStreamWriter的子类

FileReader，字符输入流

//FileReader不能指定编码格式
FileReader fr = new FileReader("文件路径");
int len;
while((len = fr.read())!=-1){
    System.out,println((char)len);    
}
fr.close();

FileWriter,字符输出流

FileWriter fw = new FileWriter("路径");
fw.write('a');
fw.close();

字符缓冲输入输出流（重点）

BufferedFileWriter，字符缓冲输出流，该流可以一次读取一行数据的方法

//写数据，BufferedWriter构造方法参数需要字符输出流
//第一步：创建文件对象
File f = new File("路径");
//第二步：创建字节流，字节流是最基本的IO流
OutputStream ops = new FileOutputStream(f);
//第三步：使用字节流，创建字符流
OutputStreamWriter osw = new OutputStreamWriter(ops);
//第四步：使用字符流创建字符缓冲流
BufferedFileWriter bw = new BufferedFileWriter(osw);
//合并的写法
BufferedFileWriter bw = new BufferedFileWriter(
    new OutputStreamWriter(
        new FileOutputStream(new File("路径"))
    )
);
//向文件写入数据
String s1 ="世界那么大";
String s2 ="钱包那么小";
//向文件写入数据
bw.write(s1);
bw.write(s2);
//文档结果---世界那么大钱包那么小
//两句合并到了一起，没有进行换行
bw.newLine();//在两句话中插入，实现换行
bw.close();

BufferedFileReader字符缓冲输入流，该流可以一次读取一行数据

//写数据。BufferedFileReader 字符缓冲输入流
//创建该对象与创建字符缓冲输出流一样，也需要从字节流开始创建
BufferedReader br = new BufferedReader(
    new InputStreamReader(
          new FileInputStream(new File("路径"))  
    )
);
//开始读取，读取一行，如果没有，会输出null
System.out.println(br.readLine());
//循环读取
String s = null;
while((s = br.readLine())!=null){
    System.out.println(s);
}

字符缓冲输出输入流读写效率非常高，一般读取文档都是使用这两个文件流
