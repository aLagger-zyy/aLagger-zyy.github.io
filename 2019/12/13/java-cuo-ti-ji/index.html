java语法

int x = 10;  
x = x++ +10;
System.out.println(x);
程序会输出（D）。 
 A.11
 B.10
 C.21
 D.20

++x：先自加，然后加入式子进行运算

x++：先加入式子运算，然后再自加

在式子 x = x++ +10 中x++先把值加入到运算中，然后自增，可是自增的时候没有赋值，所以被舍弃掉

(多选题)下列关于成员变量默认值的描述中，正确的是（ABD ）。 
 A.byte类型的数据默认值是0
 B.int类型的数据默认值是0
 C.long类型的数据默认值是0
 D.float类型的数据默认值是0.0f

(多选题)下列关于JDK的说法正确的是（AC ） 
 A.开发一个Java程序，需要完整的JDK
 B.运行一个Java程序，需要完整的JDK
 C.运行一个Java程序，只需要JDK中的JRE
 D.以上说法都不对

使用Java之前 ： 安装JDK
JDK : Java运行的环境 JVM（Java虚拟机） + JRE Java所依赖的类库
Java跨平台

运行Java程序 .java
使用javac命令将Java文件编程成.class的字节码文件
再使用 java 命令 运行.class字节码文件 （JVM运行）

那么为什么要将java文件编程成.clsaa的字节码文件呢？

因为 程序 = 数据结构+算法，程序是程序员能看懂的代码，而计算机看不懂，计算机要将程序转换成程序能看懂的字节码件。

(多选题)下列对Java变量的使用说法正确的是（ CD） 
 A.声明变量可以指定数据类型，也可以不指定
 B.变量名采用任意命名的方式，尽量做到见名知意
 C.局部变量声明后，要为其赋一个确定的初值后再使用
 D.变量是指在内存中开辟的存储空间中存放的运算过程中需要用到的数据。

变量的命名规则（法律：必须遵守的）**

只能以字母下划线或者$开始，不能包含特殊字符，不能使用Java关键字

变量的命名规范 （道德：约束）

变量名要见名知意
驼峰命名法： 首字母小写，多个单词组成，从第二个单词首字母大写。
比如：开始游戏的变量 startGame

(多选题)变量定义如下：   
int i = 128;
下列赋值语句正确的是：（ACD）。
 A.int j = i;
 B.short s = i;
 C.short s = 128;
 D.long l = i;

自动类型转换和强制类型转换
自动类型转换 ：
将像范围的数据转换成大范围的数据 ，会发生自动类型转换（隐式类型转换）
比如 ： byte -> int
强制类型转换：

​ 将大范围的数据转换成小范围的数据，需要强制类型转换（显示类型转换）

​ 比如： int -> byte

(多选题)如下变量的定义，double d = 1.5;int x=1,y=2,z=3;则正确的switch语句是（ AC）
 A.switch((int)d){ 
case 1:System.out.println(x);break; 
case 2:System.out.println(y);break; 
}
 B.switch(d){ 
case 1.5:System.out.println(x);break; 
case 2.0:System.out.println(y);break; 
}

switch括号中的值，可以是什么类型的？
jdk1.7以前 ： byte,short,char,int ，枚举
jdk1.7以后 ：除了上面的几种类型，还可以放String

switch 的运行机制：
会根据括号中的值，去匹配case中的值，一旦匹配到了值，就会从当前的case语句开始一直往下执行
直到遇到break或者switch语句为止。
